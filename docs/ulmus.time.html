ulmus.time

Creates signals useful when dealing with time.

delay

(delay t s-$)

Takes a time t in millis and a signal s-$. Returns a new signal that reflects the value of s-$ delayed by t.

every

(every interval)

Returns a signal that returns the current universal epoch time every interval milliseconds.

fps

(fps hz)

Returns a signal that fires hz times per second.

fps-when

(fps-when hz when-$)

Returns a signal that fires hz times per second when when-s is truthy.

percentage-duration

(percentage-duration t s-$)

Takes a time in millis t and a signal s-$. Returns a signal that moves from 0-1 as the lifetime of s-$ approaches t.

since

(since t s-$)

Takes a time in millis t and signal s-$. Returns a signal of true or false indicating if t has passed since the last value on s-$.

timeout

(timeout t)

Creates a signal that fires a signal value after timeout millis

with-lifetime

(with-lifetime s-$)

Takes a signal and returns a new signal of [total-time-signal-has-existed latest-value]

with-timestamp

(with-timestamp s-$)

Takes a signal and returns a new signal of [current-epoch-time latest-value]
